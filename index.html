Available pages

Get all available pages since a specific point in time

GET /{city}/{language}/wp-json/extensions/v0/modified_content/pages

Required parameters

Name /type

Description /example

Constraints

since
string

Attribute to specify the time range

"2017-05-10T11:30:00Z"

    cURL

    curl -X GET "http://cms.integreat-app.de/{city}/{language}/wp-json/extensions/v0/modified_content/pages?since=2017-05-10T11:30:00Z" \
     -H "Content-Type: application/json"
